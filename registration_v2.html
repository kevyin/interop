Registration Version 1

This class provides an interface to reading the registration metric file:

  • InterOp/RegistrationMetrics.bin
  • InterOp/RegistrationMetricsOut.bin

The file format for registration metrics is as follows:

Header

illumina::interop::io::metric_format_stream (Class that parses this information) 

    byte 0: version number
    byte 1-2: record size (uint16)

Extended Header

illumina::interop::io::generic_layout<registration_metric, 1> (Class that parses this information) 

    byte 3: channel count (uint8)
    byte 4: region count (uint8)

n-Records

illumina::interop::io::layout::base_cycle_metric (Class that parses this information) 

    2 bytes: lane number (uint16)
    2 bytes: tile number (uint16)
    2 bytes: cycle number (uint16)

illumina::interop::io::generic_layout<registration_metric, 1> (Class that parses this information) 

    4*channelCount*6 bytes: affine transform for each channel (float32)
    4*channelCount*3*regionCount bytes: sub region offset for each region and each channel (float32)
